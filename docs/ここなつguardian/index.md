# ここなつ guardian

**ここなつ guardian** は Discord サーバー向けの多機能セキュリティ Bot です。  
コンテンツフィルタ・スパム検知・レイド対策・Nuke保護など、サーバーを守るための機能を一元管理できます。

---

## 主な機能

| カテゴリ | 機能 |
|---|---|
| **コンテンツフィルタ** | 行数・文字数・メンション・添付・リンク・キーワード制限 |
| **スパム検知** | 単独スパム（高速連投）・協調スパム（クラスター） |
| **レイド対策** | ローカル Anti-Raid・グローバルレイド検知・招待バースト検知 |
| **招待フィルタ** | 外部サーバーへの招待リンク禁止 |
| **Nuke保護** | 大量チャンネル削除・ロール削除・処罰の検知 |
| **ポイント処罰** | 違反ポイント蓄積による自動タイムアウト・キック・バン |
| **ブラックリスト** |キーワード・リンクドメインのブラックリスト管理 |
| **参加者スクリーニング** | アカウント年齢・アバターなし の自動キック |
| **Honeypot** | 隠しチャンネルへのアクセス検知 |

---

## 必要な Bot 権限

guardian が正常に動作するために以下の権限が必要です。

- メンバーを管理（スクリーニング・キック処理）
- メンバーをキック
- メンバーをBAN
- メッセージを管理（削除処理）
- チャンネルを管理（Nuke保護・ログチャンネル自動作成）
- ロールを管理（タイムアウト処理）
- Webhookを管理（ログ送信・Webhookスパム検知）
- 招待リンクを管理（招待バースト検知）
- 監査ログを表示（Nuke保護・Ghost Ping検知）

:::warning ロール階層
guardian のロールは、操作対象ロールよりも **上位** に配置してください。  
ロール階層が不十分な場合、Bot はパッシブモードで動作し、処罰を保留します。
:::

---

## クイックスタート

1. Bot をサーバーに招待し、必要な権限を付与する
2. `/guardian enable` でセキュリティを有効化する
3. ログチャンネルを設定する（未設定時は自動作成）: `/guardian set-log #チャンネル`
4. 必要な機能を `/guardian toggle` でオンにする
5. 各種閾値・ポイント設定を行う

詳細は [機能一覧](./features/overview) をご覧ください。
