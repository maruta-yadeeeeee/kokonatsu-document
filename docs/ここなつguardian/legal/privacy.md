# プライバシーポリシー

最終更新日: 2026年4月8日

本プライバシーポリシーは、ここなつ guardian（以下「本 Bot」）が収集・保存・利用するデータについて説明するものです。

---

## 第1条 収集するデータ

本 Bot は以下のデータを収集・保存します。

### サーバー設定データ

| データ | 目的 |
|---|---|
| サーバー ID（guild_id）| 設定をサーバーごとに管理するため |
| 各種設定フラグ・閾値 | セキュリティ機能の動作設定 |
| ホワイトリスト（ユーザーID・ロールID・Bot ID）| スキャン除外の判定 |
| ブラックリスト（キーワード・ドメイン）| コンテンツフィルタの動作 |
| ログチャンネル ID・Honeypot チャンネル ID | アラート送信先の特定 |

### 違反・ポイントデータ

| データ | 目的 |
|---|---|
| ユーザー ID | 違反ポイントの紐付け |
| 違反タイプ・発生日時 | ポイント計算・処罰判定 |
| 累積ポイント | 自動処罰の閾値チェック |
| 処罰履歴 | 処罰ログの記録 |

### セキュリティログ

| データ | 目的 |
|---|---|
| イベントタイプ（スパム・レイド等）| セキュリティ監査・管理 |
| 関連ユーザー ID・チャンネル ID | 検知内容の記録 |
| 検知詳細（メッセージ内容の一部、最大200文字）| 管理者への報告 |
| タイムスタンプ | イベント時刻の記録 |

---

## 第2条 収集しないデータ

本 Bot は以下のデータを **収集・保存しません**。

- メッセージの全文（ログには最大 200 文字のプレビューのみ記録）
- ユーザーの個人情報（氏名・メールアドレス・電話番号等）
- 音声・画像・動画データ
- ユーザーの Discord 設定・プロフィール情報
- **ユーザー個人へのDMは一切送信しません**

---

## 第3条 データの利用目的

収集したデータは以下の目的にのみ使用します。

1. セキュリティ機能の動作（スパム・レイド・コンテンツ違反の検知・処罰）
2. サーバー管理者（オーナー）へのセキュリティアラート送信
3. 違反ポイントの管理と自動処罰の実行
4. 設定のバックアップ・復元

---

## 第4条 データの共有・第三者提供

本 Bot は収集したデータを以下の場合を除き、第三者に提供しません。

- **法令に基づく開示要求**: 法的義務に従い必要な範囲で開示する場合
- **Discord API**: Discord の規約・ポリシーに従いデータが処理される場合

クロスサーバー相関機能（`cross_server_enabled`）を有効にした場合、同機能を有効にしている他サーバーとの間でスパムメッセージのハッシュ値（内容そのものではない）が共有されます。これはスパム検知の精度向上を目的とするものです。

---

## 第5条 データの保存期間

| データ種別 | 保存期間 |
|---|---|
| サーバー設定 | Bot がサーバーから退出するまで |
| 違反ポイント・処罰履歴 | 設定されたポイント減衰設定に従い更新 |
| セキュリティログ | サーバー管理者が削除するまで（自動削除なし）|
| 外部招待キャッシュ | `external_invite_ttl_days` 設定値（デフォルト 3 日）|
| メモリ上の一時データ | Bot 再起動時に消去 |

Bot がサーバーから退出（kick/ban）された場合、そのサーバーに関するメモリ上のデータはすぐに消去されます。データベース上のデータについては、管理者からの要請により削除対応します。

---

## 第6条 セキュリティ

収集されたデータは PostgreSQL データベースに保存されます。データベースへのアクセスは Bot 実行環境のみに限定されており、不正アクセスを防ぐための措置が講じられています。

---

## 第7条 お子様のプライバシー

本 Bot は 13 歳未満の方を対象とするサービスではありません。Discord の利用規約に従い、13 歳未満の方の利用はできません。

---

## 第8条 ポリシーの変更

本ポリシーは必要に応じて更新されることがあります。重要な変更がある場合は可能な範囲でお知らせします。最終更新日は本ページ上部に記載されています。

---
