# ここなつ ドキュメント

ここなつが提供するサービスの公式ドキュメントへようこそ。  
各機能の詳細・利用方法・法的文書をこちらからご確認いただけます。

---

## サービス一覧

### 🔐 ここなつアカウント
Discord SSOとメール/パスワードを統合した共通認証基盤。多要素認証（MFA）・デバイス管理・OAuth2 IdPとして機能します。

[詳しく見る →](./function/kokonatsu_account)

---

### 💰 ここなつ経済システム
仮想通貨 **kk** と AMM型仮想資産 **kkc**。ランク制度・課税・ショップが連動した仮想経済システムです。

[詳しく見る →](./function/kokonatsu_economy)

---

### 🐾 ここペット
kkを消費してペットを育成できる仮想ペットシステムです。経済システムと連動しています。

[詳しく見る →](./function/kokopets)

---

### 🗂️ ここドライブ
クラウドストレージ。共有リンク・E2E暗号化金庫・使い捨てURLを提供します。

[詳しく見る →](./function/kokodrive)

---

### 📋 ここぼーど
Discordサーバーを掲示板に登録して宣伝できるサービス。Bump・レビュー・タグ検索に対応しています。

[詳しく見る →](./function/kokoboard)

---

### 💬 グローバルチャット
複数のDiscordサーバー間でメッセージをリレー同期する機能。Webhookを使ってリアルタイムに転送されます。

[詳しく見る →](./function/kokoglobalchat)

---

### 🎰 ここカジノ
仮想通貨（kk）のみを使用したカジノゲーム。**Provably Fair（公正性証明）** 方式を採用しています。

[詳しく見る →](./function/kokocasino)

---

### 🔗 OAuth2 / 外部連携
ここなつアカウントで外部サービスにログインできるOAuth2 / OpenID Connect（OIDC）プロバイダ機能。Authorization Code + PKCEに対応。

[詳しく見る →](./function/oauth2)
